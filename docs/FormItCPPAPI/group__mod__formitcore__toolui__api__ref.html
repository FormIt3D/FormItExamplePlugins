FORMIT_CORE_EXPORT void FormIt::ToolUI::Paintbrush ( size_t  materialID,
bool  bBackSide = false,
bool  bPickerMode = false 
)

Invokes the paintbrush tool using the id of the formit material to paint with.

If WSM::INVALID_ID is passed, the tool uses the default material (i.e. removes the material from the object). bBackSide determines whether the front or backside of a face is painted. If bPickerMode is true, the tool first performs a single face selection, which sets the material to paint with. After that it continues in paint mode with that material.

HAS SC

Parameters
[in]materialID
[in]bBackSide
[in]bPickerMode
