FORMIT_CORE_EXPORT I18nString FormIt::AutoSave::GetExistingBackupFile ( const I18nString &  filename)

Checks whether or not a backup file exists for the given path.

If the input file is an axm file and an axm backup file exists, the path to the backup file is returned. An empty string is returned if:

  • The input file is not an axm file
  • A backup file does not exist for the given axm file
    Parameters
    [in]filename
