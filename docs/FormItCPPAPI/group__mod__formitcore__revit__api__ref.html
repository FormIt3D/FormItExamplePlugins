Import a file, assumed to be exported from revit.

This will generate one or more groups and their instances. Create a new layer with the given name and place the instances on it Note : This currently is meant to be executed at startup when Formit is started from the LauchFormIt Revit Add-in with a file that was exported fomr Revit. Eventually this code can be re-used when a new or updated model from Revit is re-exported and imported in the same session. How exactly the new and previously
imported models are treated in FormIt remains to be determined.
