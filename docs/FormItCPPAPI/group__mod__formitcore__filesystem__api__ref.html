FORMIT_CORE_EXPORT std::vector<std::string> FormIt::FileSystem::GetContentsOfDirectory ( const std::string &  directoryPath,
const std::vector< std::string > &  extensionFilters 
)

Returns all the files and folders in the given directory.

This it a flat listing (i.e. no items in subdirectories are included). If the extensionFilters vector is not empty, only files which have any of the extensions in the vector are returned. The filter "directory" is a special filter and means that directories are included in the result. Extensions should be in all lower case.

Parameters
[in]directoryPath
[in]extensionFilters
