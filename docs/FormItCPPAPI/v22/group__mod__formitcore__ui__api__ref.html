◆ ShowNotification() [2/2]

FORMIT_CORE_EXPORT FormIt::NotificationHandle FormIt::UI::ShowNotification ( WSM::Result const &  wsmres,
I18nString const &  message = {},
bool  bSelectObjectsFromResult = false,
int  nTimeout = 0,
LinkCallback  linkCallback = {} 
)

Display a notification message to the user.

This function determines whether the notification is an error or warning depending on the status of the Result object. If it is good and not a warning, it returns without doing anything. If the message parameter is not empty, then that message appears first, followed by the string from the Result on the next line. If bSelectObjectsFromResult is true, then any objects referenced by wsmres are added to the current selection. The nTimeout parameter is the time in ms that the notification should be displayed. A value of 0 means to use the default timeout.

Returns
A handle to the notifiction dialog that can be used to control the dialog via other FormIt APIs Called when a hyper link is clicked with the href text passed to linkCallback
