◆ UpdateRenderer()

FORMIT_CORE_EXPORT void FormIt::Graphics::UpdateRenderer ( size_t  nHistoryID)

This is called to force a renderer for a specific history to update for Dynamo.

The update doesn't happen with the kLoadComplete broadcast since the reference histories already exist. Merely broadcasting again is potentially expensive due to all the listeners for kLoadComplete (for example, we don't want the material widget to update in this case, etc).

Parameters
[in]nHistoryID
